A Book Bucket List App

Have books that you've read or want to read and wish you could add them to your own personal library? Now you can with Book Lit.

Save Your Books

With these four libraries you are able to store an infinite number of books and move them from one list to another with a simple button click.

Simply Search And Add

Search for the book you want to add and select which category to save it to and your done. That's it your book has been added

Move Any Book

Select the book that you want to move and select which category you would like to move it to.

Deleting A Book

Not only can you move a book but you can them as well.
